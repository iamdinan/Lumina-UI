# Lumina UI

React frontend for **Lumina**, a TV series tracker. Search for series, add
them to your personal list, mark episodes watched, and view watch stats.

Built with React + Vite, Tailwind CSS v4, DaisyUI, TanStack Query, React
Router, React Hook Form + Zod, Axios, and `lucide-react`.

## Quick start

```bash
npm install
npm run dev
```

See `AGENTS.md` for coding conventions, `docs/ARCHITECTURE.md` for the
folder layout and data flow, and `docs/DECISIONS.md` for design rationale.
